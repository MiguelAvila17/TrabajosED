# Proyecto Gestor-BST-Consola - Día 12

## Uso
- Compila y ejecuta con Code::Blocks abriendo el `.cbp`.
- Comandos disponibles: `mkdir`, `touch`, `mv`, `rm`, `ls`, `search`, `export`, `exit`.
- `touch <nombre> <contenido>` admite espacios en contenido.

## JSON
- En Día 4 se guarda y carga desde `filesystem.json`.

## Estructuras
- Árbol general con `Nodo`.
- Trie para autocompletado.
- Hash map para búsqueda exacta.
