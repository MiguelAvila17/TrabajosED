# Presentación

- Objetivo: simular un sistema de archivos jerárquico.
- Estructuras: Árbol general, Trie, Hash map.
- Persistencia: JSON local `filesystem.json`.
- Comandos: `mkdir`, `touch`, `mv`, `rm`, `ls`, `search`, `export`.
- Métricas: tamaño y altura del árbol (se integran en la versión final).
- Demostración: crear y listar nodos; autocompletar; exportar preorden.
- Retroalimentación: riesgos, escalabilidad, mejoras futuras.
